## Section 2: Quick Actions - Proofread (1:15 - 2:00)

### Visual
- Click the AI button
- Menu opens showing all 4 sections

### Narration

> "The menu is organized into four sections: Quick Actions, Rewrite, Format, and Compose. Let's start with the most useful feature - Proofread."

### Action
- Hover over "Proofread" option (shows checkmark icon)
- Click "Proofread"

### Visual
- Loading indicator appears briefly
- Preview window opens showing:
  - Original text on left
  - Corrected text on right
  - Differences highlighted in orange

### Narration

> "Within two seconds, we get a preview showing the original and corrected text side-by-side. Notice how 'your' becomes 'you're', 'discused' becomes 'discussed', and it even capitalizes 'John'. The changes are highlighted in orange so you can review them."

### Action
- Click "Accept" button

### Visual
- Text in email is replaced with corrected version
- Preview window closes

### Narration

> "Click Accept, and the corrected text replaces the original instantly. That's it - your text is proofread!"

### On-Screen Text
- "Fixed 4 errors"
- "2 seconds"

## Section 3: Rewrite Options (2:00 - 3:15)

### Visual

### Narration

> "The Friendly tone keeps it casual but more polished - great for team communication."

### Narration

## Post-Production Notes

### Editing

1. **Pacing**: Keep each section moving quickly
2. **Transitions**: Smooth fade between sections
3. **Highlights**: Circle or arrow for important UI elements
4. **Speed**: Consider 1.1x speed for efficiency
5. **Music**: Subtle background music (optional)

### Graphics Overlay

- Section titles appear at start of each section
- Keyboard shortcuts shown in corner when mentioned
- Performance stats (timing, accuracy) as overlays
- Icons for each action type

### Voiceover Tips

- Clear, friendly tone
- Moderate pace (not too fast)
- Emphasize key benefits
- Pause briefly between sections

### B-Roll Suggestions

- Quick cuts showing different applications
- Close-up of button appearing
- Menu opening/closing
- Preview window highlights
