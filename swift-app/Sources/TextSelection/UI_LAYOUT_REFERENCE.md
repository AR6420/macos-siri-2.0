## Responsive Behavior

The panel adjusts to content but maintains constraints:

```
Minimum height: ~300px (all sections visible)
Maximum height: 500px (prevent overflow)
Fixed width: 320px (optimal readability)

If content exceeds max height:
- ScrollView wraps content (not currently needed)
- Consider collapsing sections (future enhancement)
```
